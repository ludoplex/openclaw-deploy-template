## How to Request Offline Materials

### General Process
1. Search online catalog for holdings
2. Contact reference desk for availability
3. Request retrieval (may take hours/days for offsite storage)
4. Some materials require appointment
5. Some require researcher credentials/letters of introduction

### NARA Research
- Create researcher card (free, on-site)
- Request records via National Archives Catalog
- Some records require advance pull requests
- Classified records require FOIA/MDR

### Interlibrary Loan (ILL)
Many rare items available via ILL through local library. Request through:
- Local library ILL desk
- WorldCat resource sharing
