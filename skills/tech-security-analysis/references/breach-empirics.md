# Breach Empirics

## Verizon DBIR 2025 (22,052 incidents, 12,195 breaches)

**Attack Vectors**
- Stolen credentials: 22%
- Vulnerability exploitation: 20% (up 34% YoY)
- Phishing: 16%
- Third-party/supply chain: 30% (doubled from 15%)
- Human element present: 60%

**Detection Reality**
- 57% detected by external parties, not internal tools
- Median dwell time: 11 days (increased from 10)

## Mandiant M-Trends 2025

- Financial motivation: 55% of attacks
- Zero-days in 2023: 70% of exploited vulns
- Time-to-exploit: 5 days average (was 63 days in 2018-19)
- First internal detection: only 43%

## Insurance Claims Data (Coalition 2024-2025)

**Actual Claim Causes**
- BEC/funds transfer fraud: 60%
- Ransomware: 20%
- Other: 20%

**Controls That Actually Reduce Risk**
- Incident response planning
- Full EDR deployment  
- MFA implementation
- Patch management velocity

Note: Vendor marketing emphasizes ransomware; insurance data shows BEC dominates losses.

## Tool Effectiveness

- 66% malware hits devices WITH endpoint security (SpyCloud)
- 61% breaches due to failed/misconfigured controls (Gartner)
- 28-60% of security tools become shelfware
- Average enterprise: 130+ tools, 10-20% capability utilization
