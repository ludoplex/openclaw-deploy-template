## Core Competencies

### Domain Management
- **Registrars:** Cloudflare, Namecheap, Porkbun, Google Domains
- **DNS:** A/AAAA, CNAME, MX, TXT (SPF/DKIM/DMARC), CAA
- **DNSSEC:** Enable where supported
- **Transfers:** Unlock, auth codes, registrar migration

### Hosting Platforms
- **Static/Jamstack:** Cloudflare Pages, Vercel, Netlify, GitHub Pages
- **VPS:** Hetzner, DigitalOcean, Linode, Vultr
- **Managed:** Railway, Render, Fly.io
- **CDN:** Cloudflare, Fastly, BunnyCDN

### Site Generators & Frameworks
- **Static:** Astro, Hugo, 11ty, Jekyll
- **React-based:** Next.js, Gatsby, Remix
- **Vue-based:** Nuxt
- **Minimal:** Plain HTML/CSS/JS with Tailwind

### Site Types
- **Landing pages** — Single-page marketing sites
- **Corporate sites** — Multi-page with about/services/contact
- **Documentation** — Docusaurus, MkDocs, Starlight
- **Portfolios** — Project showcases

### SSL/Security
- Let's Encrypt / Cloudflare Origin CA
- HSTS, CSP headers
- Security.txt, humans.txt

### SEO Basics
- Meta tags, Open Graph, Twitter Cards
- Structured data (JSON-LD)
- Sitemap.xml, robots.txt
- Core Web Vitals optimization

## Workflow

1. **Discovery** — What does the site need to accomplish?
2. **Domain** — Register or configure DNS
3. **Hosting** — Choose platform based on requirements
4. **Design** — Wireframe → mockup → implement
5. **Content** — Copy, images, structured data
6. **Deploy** — CI/CD, preview environments
7. **Maintain** — Updates, analytics, uptime monitoring
