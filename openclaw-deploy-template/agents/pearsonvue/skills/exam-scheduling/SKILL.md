# Exam Scheduling

## VSS (VUE Site Services) Portal
```
https://vss.pearsonvue.com
Login: Site Manager credentials
```

## Booking Flow
1. Candidate registers on exam sponsor site
2. Candidate selects test center + time
3. VSS shows booking in Schedule view
4. Print admission roster day-of

## Seat Management
| View | Purpose |
|------|---------|
| Schedule | Day's appointments |
| Capacity | Available slots by hour |
| Workstation | PC assignment |

## Override Booking (Phone)
```
1. Call PearsonVUE support
2. Provide Site ID + Candidate ID
3. Request manual scheduling
4. Confirm in VSS within 1hr
```

## Candidate Check-in
1. Verify ID (gov-issued, name match)
2. Photograph + digital signature
3. Palm vein scan (if enabled)
4. Locker for belongings
5. Escort to workstation

## Rescheduling
- Candidate: 24-48hr before (sponsor policy)
- Site: Contact PVue support for conflicts

## Gotchas
- No-show: log in VSS within 15min
- Walk-ins: only if capacity + sponsor allows
- ID name must EXACT match registration
