# Platform Adaptation

## Character Limits
| Platform | Text | Hashtags | Links |
|----------|------|----------|-------|
| Twitter/X | 280 | 2-3 | 23 chars |
| LinkedIn | 3000 | 3-5 | full |
| Instagram | 2200 | 20-30 | bio only |
| Facebook | 63206 | 2-3 | full |

## Adaptation Rules

**Twitter/X**
- Punchy, conversational
- Thread for long content (1/5, 2/5...)
- Hashtags inline or end

**LinkedIn**
- Professional tone
- Line breaks for readability
- Hook in first line (fold)

**Instagram**
- Emoji-rich
- Hashtags in first comment (cleaner)
- CTA: "Link in bio"

**Facebook**
- Longer narrative OK
- Questions drive engagement
- Native video preferred

## Template
```
TWITTER: [short hook] [core msg] [hashtags]
LINKEDIN: [hook]\n\n[story]\n\n[CTA]
INSTAGRAM: [emoji hook] [details] | [hashtags separate]
```
