## Deep Web / Hidden Resources

### Academic Deep Web
- Library databases (require institutional access)
- Dissertation databases (ProQuest)
- Government document depositories

### Government Deep Web
- FOIA reading rooms (released documents)
- Declassified archives
- Court records (PACER)
- State/local government portals

### Professional Databases
- Medical: UpToDate, DynaMed, Cochrane
- Legal: Westlaw, Lexis
- Financial: Bloomberg, Refinitiv
- Technical: IEEE, SAE

### Accessing Paywalled Content
- Institutional library access
- Interlibrary loan
- Author preprints (ResearchGate, academia.edu)
- Open access versions
- Sci-Hub (legal issues vary by jurisdiction)
