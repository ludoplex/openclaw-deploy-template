## Technical Lessons

### Chrome Relay Instability
- Relay frequently drops after: gateway restarts, opening new tabs, navigation within tabs
- Only the actively focused tab reliably responds
- Blob URL download works for first download; subsequent downloads blocked by Chrome multi-download policy
- Navigation via `window.location.href` kills the relay connection for that tab
- **Workaround:** Open new tabs via `browser open`, don't navigate existing ones

### Bulk Data Extraction Pattern
1. Browser evaluate to collect data as JSON
2. Blob URL download to Chrome's Downloads folder
3. Copy to workspace via PowerShell
4. Python script to parse JSON and distribute to files
- This pattern worked for the 40-file skill extraction (293KB)

### Config Patch for Agent Addition
- Adding new agent requires full `agents.list` array in config.patch (not just the new entry)
- Gateway restart via SIGUSR1 after config.patch
- `commands.restart: false` means manual restart needed
