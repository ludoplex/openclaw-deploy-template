## Gateway Crash Fix
- Rapid config.patch calls (3 in a row for TTS tuning) caused gateway crashes
- Each patch triggered auto-restart via commands.restart: true
- Another LLM diagnosed it — shutdown was timing out
- Fix: Set commands.restart: false — config edits no longer auto-restart gateway
- Lesson: Batch config changes or disable auto-restart when tuning
