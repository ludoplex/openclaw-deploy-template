## Key Decisions Made Today
- **UNIX-first AI philosophy**: Every AI concept maps to a UNIX daemon. Daemons communicate via sockets, pipes, signals, files.
- **Two-tier product**: cosmo-bsd base (~150MB/512MB RAM) and cosmo-bsd-ai (with llamafile + daemons ~4-8GB/4GB+ RAM)
- **Agents as FreeBSD jails**: Each agent = jail with filesystem mailboxes, kqueue for delivery
- **Toybox as "cheat code"**: Single APE binary fills almost all userland gaps
- **cosmo-bsd repo is PRIVATE**: Competitive advantage for DSAIC/MHI GovCon
- **Icecat over Etilize**: Free (25.8M+ datasheets) vs paid. Same IT coverage.
- **Amazon PA-API dying April 2026**: Don't build on it
- **Ingram Micro first for API integration**: Best docs, SDKs, sandbox, webhooks
- **commands.restart: false**: Prevents gateway crash from rapid config edits
- **Cosmo-python for shipping, system Python for dev**: Can't pip install into cosmo-python
