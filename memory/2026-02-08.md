## Lessons Learned

### Token Format Confusion
Zoho grant codes look like: `1000.xxx.xxx` (two hex sections)
Zoho client IDs look like: `1000.XXXXXXXX` (shorter, one section)
Zoho client secrets look like: `xxxxxxxxxxxxx` (no periods) OR `1000.xxx.xxx` (varies by account!)

Vincent's Self Client had a client secret in the `1000.xxx.xxx` format, which I mistook for a grant code. Cost 10+ minutes. Always check the downloaded `self_client.json` files in Downloads folder.

### Google OAuth External Apps
External apps in "Testing" mode require adding users to test user list before they can authorize. Navigate to: Google Cloud Console ‚Üí Auth Platform ‚Üí Audience ‚Üí Add users

### Chrome Relay Recovery
After "tab not found" errors, use `browser action: "open"` with a new URL rather than trying to navigate stale targetIds.
